# 💱 Currency Converter App

A simple and responsive **Currency Converter Web Application** built using **HTML, CSS, and JavaScript**.

This app allows users to convert currencies in real-time using exchange rate data and provides a clean and user-friendly interface.

---

## 📌 Project Overview

The Currency Converter App helps users easily convert one currency to another by selecting the desired currencies and entering an amount.

It demonstrates:
- Frontend development using JavaScript  
- API integration for real-time data  
- DOM manipulation  
- Event handling  

---

## ✨ Features

- 💱 Convert between multiple currencies  
- 🔄 Real-time exchange rate calculation  
- 🔁 Swap currencies (if implemented)  
- ⚡ Instant results  
- 📱 Responsive design
